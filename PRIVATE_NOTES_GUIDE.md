# 🔐 Cara Melihat Notes Private di Journal App

## 📋 **Panduan Lengkap Mengakses Notes Private**

### **1. Membuka Semua Notes (Default)**
- **Buka journal app** di http://localhost:3000
- **Semua notes** (private & public) akan muncul secara default
- **Tidak ada filter** yang aktif saat pertama kali membuka

### **2. Mengidentifikasi Notes Private vs Public**

**🔒 Indikator Notes Private:**
- **Tidak ada ikon share** (🔗) di card note
- **Tidak ada ikon globe** (🌐) 
- **Tampilan normal** tanpa badge publik

**🌐 Indikator Notes Public:**
- **Ada ikon share** (🔗) di card note
- **Ada ikon globe** (🌐) dengan warna hijau
- **Badge "Public"** terlihat di card

### **3. Cara Filter Notes**

#### **🔍 Berdasarkan Search:**
1. **Gunakan search bar** di bagian atas sidebar
2. **Ketik kata kunci** dari note yang dicari
3. **Akan muncul** notes yang mengandung kata tersebut (private & public)

#### **😊 Berdasarkan Mood:**
1. **Klik mood filter** di sidebar (happy, tired, overthinking, missing, calm)
2. **Akan muncul** notes dengan mood tersebut (private & public)
3. **Klik "All"** untuk kembali ke semua notes

### **4. Cara Membuat Notes Private**

#### **📝 Saat Membuat Note Baru:**
1. **Klik "New Note"** di floating button
2. **Isi form** (title, content, mood, date)
3. **Pastikan toggle "Public" dalam posisi OFF** (default)
4. **Klik "Save Memory"**

#### **🔧 Mengubah Note Public ke Private:**
1. **Temukan note public** (ada ikon 🌐)
2. **Klik ikon globe** (🌐) di card note
3. **Note akan berubah** menjadi private
4. **Ikon globe akan hilang**

### **5. Cara Melihat Detail Notes Private**

1. **Klik card note** yang diinginkan
2. **Note akan terbuka** di detail view
3. **Private notes** tidak memiliki share options
4. **Aman dan pribadi** hanya untuk kamu

### **6. Fitur Tambahan untuk Notes Private**

#### **🌟 Menjadikan Special Note:**
1. **Klik ikon bintang** (⭐) di card note
2. **Note akan menjadi** special note
3. **Hanya satu special note** yang bisa ada
4. **Akses cepat** dari sidebar

#### **📅 Timeline View:**
1. **Klik "Timeline"** di sidebar
2. **Lihat semua notes** dalam bentuk timeline
3. **Private & public** notes terlihat semua
4. **Urutan kronologis** berdasarkan tanggal

#### **🎲 Random Memory:**
1. **Klik "Random"** di sidebar
2. **Akan muncul** note random (bisa private atau public)
3. **Kejutan memories** dari masa lalu

### **7. Keamanan Notes Private**

#### **🔒 Fitur Keamanan:**
- **Tidak bisa di-share** ke publik
- **Tidak ada link public** untuk diakses orang lain
- **Hanya kamu** yang bisa melihat dan edit
- **Tidak muncul** di public notes list

#### **🛡️ Privacy Protection:**
- **Server-side validation** untuk akses
- **Database filtering** untuk query
- **No public URLs** untuk private notes
- **Encrypted storage** di Supabase

### **8. Tips Mengelola Notes Private**

#### **📝 Organisasi:**
- **Gunakan mood filter** untuk kategorisasi
- **Search dengan keywords** yang spesifik
- **Buat special note** untuk memories penting
- **Gunakan timeline** untuk melihat perkembangan

#### **🔍 Pencarian Efektif:**
- **Cari berdasarkan tanggal** (format: YYYY-MM-DD)
- **Cari berdasarkan mood** keywords
- **Cari frasa spesifik** dari content
- **Gunakan kombinasi** search + mood filter

### **9. Troubleshooting**

#### **❓ Note Tidak Muncul:**
- **Cek search bar** - kosongkan dulu
- **Cek mood filter** - klik "All"
- **Refresh halaman** dengan F5
- **Cek koneksi internet**

#### **❓ Note Tetap Public:**
- **Klik ikon globe** sekali lagi
- **Tunggu loading** selesai
- **Refresh halaman**
- **Coba buat note baru**

### **10. Quick Reference**

| **Aksi** | **Cara** | **Hasil** |
|---------|----------|-----------|
| **Lihat semua notes** | Buka app | Private + Public |
| **Cari note spesifik** | Search bar | Filtered results |
| **Filter by mood** | Klik mood | Mood-specific notes |
| **Buat note private** | New Note + Public OFF | Private note |
| **Ubah ke private** | Klik 🌐 icon | Note menjadi private |
| **Lihat detail** | Klik card | Detail view |
| **Jadikan special** | Klik ⭐ icon | Special note |

## 🌟 **Best Practices**

### **Untuk Privacy Maksimal:**
1. **Selalu pastikan** toggle Public OFF saat membuat note sensitif
2. **Gunakan keywords** yang spesifik untuk pencarian
3. **Regular check** notes yang public vs private
4. **Hapus notes** yang tidak needed

### **Untuk Organisasi:**
1. **Gunakan mood** untuk kategorisasi otomatis
2. **Buat special note** untuk memories penting
3. **Gunakan timeline** untuk refleksi diri
4. **Search reguler** untuk menemukan lama notes
